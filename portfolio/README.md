# Modern Minimalist Portfolio

A modern, minimalist, and responsive portfolio website built with React, TypeScript, and Vite. Features a dark and orange color theme.

## Features
- ⚡️ Built with Vite, React, and TypeScript
- 🎨 Minimalist, modern design
- 🌙 Dark mode with orange accent
- 📱 Fully responsive

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) to view it in your browser.

## Customization
- Update your name, about, projects, and contact info in `src/App.tsx`.
- Customize styles in `src/index.css` and `src/App.css`.
